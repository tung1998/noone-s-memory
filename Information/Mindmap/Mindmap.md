# Mindmap dòng thời gian Memnet

Tài liệu này là node gốc dạng chữ của thế giới *No One's Memory*. Trong `Information/Mindmap/`, mọi file đều được xem là node; khác nhau chủ yếu ở tính chất và folder chứa chúng.

Graph thuần liên kết nằm tại [Mindmap.json](./Mindmap.json). File JSON chỉ chứa `id`, `file`, và `relationships` giữa các node; không chứa tóm tắt nội dung.

| Lớp | Mục đích | Đường dẫn |
| --- | --- | --- |
| Root node | Nhìn nhanh toàn bộ dòng thời gian và quan hệ giữa các node | [Mindmap.md](./Mindmap.md) |
| Graph node | Xem graph liên kết thuần giữa các node | [Mindmap.json](./Mindmap.json) |
| Timeline node | Mỗi mốc lớn có một file riêng để đọc nhanh | [Timeline/](./Timeline/) |
| Lore node | Giải thích tổ chức, công nghệ, địa điểm, khái niệm | [Nodes/](./Nodes/) |
| Character node | Tóm tắt hồ sơ và vai trò của từng nhân vật | [Character/](./Character/) |
| Project node | Tóm tắt premise, direction, và writing rules cấp dự án | [Project/](./Project/) |

| Giai đoạn | Mốc chính | Hệ quả |

| 2075-2080 | Nguy cơ sụp đổ lần hai và lựa chọn cuối cùng | Xung đột giữa kiểm soát, tự do, và ký ức thật |

## Luồng quan hệ giữa các node

```text
Ký ức số hóa cực độ
-> Đại Sụp Đổ
-> Kỷ nguyên trắng ký ức
-> Cognitive Survivalists tạo Memnet
-> Memlink + MIC + quản trị ký ức toàn cầu
-> Kiểm soát xã hội bằng ký ức
-> Xã hội Memnet / tự xóa ký ức / Thanh tra ký ức / Ký ức Tự Do / Vùng Trắng / nhà trẻ không người nhớ
-> Bạch Phong + Mira từ MIC Zone 4 lần theo sai lệch
-> Chợ đen ký ức + Mnemonic Shard + Seiromemdas
-> Hé lộ sự thật về Đại Sụp Đổ và tương lai của Memnet
```
