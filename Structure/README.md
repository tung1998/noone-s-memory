# Cây cấu trúc chương

## Mục đích

- Thư mục này dùng để giữ bản tóm tắt triển khai của từng chương.
- Mỗi file là một bản rút gọn theo hướng viết tiếp, không thay thế `Raw/` hay `Edited/`.
- Nội dung ưu tiên vai trò chương, điểm lệch, điểm xoay, và dư âm cuối để tiện mở rộng hoặc viết lại.

## Cách dùng

- Đọc file tương ứng trước khi triển khai hoặc sửa chương.
- Nếu mạch chương thay đổi lớn, sửa file tóm tắt này trước hoặc cùng lúc với chương.
- Giữ tóm tắt ngắn, cụ thể, bám đúng những gì chương cần làm trong toàn truyện.
