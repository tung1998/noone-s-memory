# Lore File Template

Dùng mẫu này khi tạo file mới trong `Information/`.

## Tên file

- Dùng tên rõ nghĩa, bám theo thực thể hoặc khái niệm.
- Ví dụ:
  - `World.md`
  - `MnemonicShard.md`
  - `Character/Bạch_Phong_hkz-746-xy36f.md`

## Mẫu nội dung

```md
# Tên mục lore

## Tổng quan

Mô tả ngắn 2-4 câu: đây là gì, vì sao nó quan trọng trong thế giới truyện.

## Vai trò trong thế giới

| Mặt | Nội dung |
| --- | --- |
| Chức năng | Nó dùng để làm gì |
| Ảnh hưởng | Nó tác động đến xã hội, nhân vật, hoặc hệ thống ra sao |
| Rủi ro | Điều gì khiến nó nguy hiểm, sai lệch, hoặc dễ bị lạm dụng |

## Liên hệ với cốt truyện

- Nó xuất hiện ở đâu.
- Liên quan đến nhân vật nào.
- Có thể mở ra bí ẩn hay xung đột gì.

## Ghi chú continuity

- Cách viết chính tả chuẩn.
- Mốc thời gian liên quan.
- File canon nào khác cần giữ nhất quán.
```

## Quy tắc dùng template

- Không cần nhồi đủ mọi mục nếu note nhỏ.
- Nhưng ít nhất phải có:
  - định nghĩa hoặc tổng quan;
  - vai trò;
  - liên hệ với truyện hoặc timeline.
- Nếu file thêm thực thể mới, nhớ cập nhật `SUMMARY.md`.
