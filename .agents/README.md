# .agents

Thư mục này dùng để gom các quy tắc và workflow dành cho Codex trong repo *No One's Memory*.

## Gợi ý dùng

- `codex-rules.md`: quy tắc làm việc chính cho Codex.
- `chapter-workflow.md`: quy trình viết hoặc sửa một chương.
- `continuity-checklist.md`: checklist chống lệch canon.
- `review-checklist.md`: checklist review bản thảo.

## Bộ file hiện có

| File | Mục đích |
| --- | --- |
| `codex-rules.md` | Rule tổng quát khi Codex làm việc trong repo |
| `chapter-workflow.md` | Các bước chuẩn khi viết chapter |
| `continuity-checklist.md` | Soát timeline, lore, nhân vật, thuật ngữ |
| `review-checklist.md` | Soát lỗi, rủi ro, pace, mystery, continuity |
| `lore-file-template.md` | Mẫu tạo file mới trong `Information/` |
| `chapter-template.md` | Mẫu phác và kiểm chương mới |
| `retcon-policy.md` | Quy tắc thay đổi canon có kiểm soát |

## Skill cục bộ

- `.agents/skills/sang-tao/`: skill ideation để nảy hướng truyện, scene node, và biến thể tình huống.
- `.agents/skills/trien-khai/`: skill mở rộng node hoặc beat thành văn xuôi dài, đúng tone và continuity.

## Lưu ý

`AGENTS.md` ở root vẫn là điểm vào trực tiếp mà Codex sẽ đọc trước. Thư mục `.agents/` dùng để tổ chức rule rõ ràng hơn cho người viết và cho các lần mở rộng sau.
